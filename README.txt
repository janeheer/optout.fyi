# optout.fyi

Immigrant data rights letter generator. A free, client-side tool that generates legally-grounded CCPA/CPRA privacy rights letters targeting AI surveillance companies involved in immigration enforcement.

Bilingual (English/Spanish). Zero data collection. No accounts, no tracking, no cookies.
